How to use this demo


The empty grid is an empty floorplan of a warehouse.

The green square represents the start point of a pick route.

First, set squares to represent racking.

Then set pick bays to determine squares that must be routed to.

After clicking find optimal route, all possible edges will be highlighted.

Then, click the optimal route card to visualise the optimal pick route.
